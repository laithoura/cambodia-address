# Cambodia Address

Convert Latitude & Longitude to address

## Installation

```
npm install @pigeon-media/cambodia-address
```

## Usage

```js

const location = { lat: 0.0, lng: 0.0 };
const result = resolve(location);
console.log(result);
```

## Execute

```bash
npx cambodia-address 100,100
```
